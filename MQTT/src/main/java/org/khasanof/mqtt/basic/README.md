# Пример MQTT Pub/Sub (Eclipse Paho)

Минимальный пример публикации и подписки на сообщения по протоколу MQTT.

## Что внутри

- **`MqttPublisher`** — клиент-издатель: подключается к брокеру `tcp://localhost:1883`, публикует строку `Hello MQTT!` в топик `test/topic` с **QoS 1**, затем отключается. Цикл повторяется несколько раз с новым подключением при каждой итерации.
- **`MqttSubscriber`** — клиент-подписчик: подключается к тому же брокеру, подписывается на `test/topic` и выводит в консоль полезную нагрузку каждого полученного сообщения.

Оба класса запускаются отдельно через `main`.

## Перед запуском

1. Должен быть доступен MQTT-брокер на **localhost:1883** (например, Eclipse Mosquitto). В проекте есть Docker Compose: `src/main/docker/paho.yml`.
2. Сначала запустите **`MqttSubscriber`**, затем **`MqttPublisher`**, чтобы увидеть входящие сообщения у подписчика.

## Зависимости

Клиент — **Eclipse Paho** (`org.eclipse.paho.client.mqttv3`).
